NoTanks is a little multiplayer game (2 to 4 players) about tanks fighting in a top-view arena.

Your objective is simple : destroy your opponents' tanks before your own gets exploded !

A short presentation :

It's a personal project that I coded in my free time, and the code is publicly available for free on GitHub as an OpenSource project.

It was made on Unity.
The development of the game is done in C#.
